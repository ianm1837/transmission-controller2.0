# Transmission Controller 2.0

4r70w OD and Lock up relay controller and gear display.

### Notes

- In order to flash fonts to SPIFFS OR use esptool to flash you must first put board in "ROM Boot Mode"
  - Short B1 to GND
  - LED should be Green
  - Press reset button
  - Remove jumper from B1/GND
  - LED should be dim purple
